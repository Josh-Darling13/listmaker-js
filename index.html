Warning...

I designed Listmaker Plus + 0.1 for my personal use for coding. Yes, it is available for public use. However, most people don't sit down to code on their phones --I don't. I designed this version for screens larger than 1000px. This program is buggy and looks sloppy under 900px (smaller screens). I know designing mobile-first is a best practice, but I feel this is a solid exception.

In the future, I'll get around to a tablet version. A mobile version will follow. I'd rather build developer features and make this useful for the majority. I'll get to the people who code on their iPhones when I can. I wrote this software with myself in mind. Helping others was a nice afterthought.
